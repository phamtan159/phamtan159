## Featured Project

### CT240 Team Task Management System

A full-stack web application for managing team projects, tasks, members, comments, notifications, and reports.

**Backend**

- Spring Boot REST API
- MongoDB data storage
- JWT authentication
- Role-based access control: Admin, Manager, Member
- Project, task, comment, notification, activity log modules
- Observer, Strategy, Factory, and Template Method patterns

**Frontend**

- Vue 3 + Vite
- Vuetify UI
- Pinia state management
- Axios API integration
- Admin and member dashboards
- Project, task, notification, profile, and report screens
